Before running the executable, make sure that the "resources" directory and its contents, and the files "shader.fs" and "shader.vs" are in the same directory as the executable "Assignment0.exe".

Then run "Assignment0.exe"

When using the program, use the GUI to move the camera and change render settings.

To load a new model, imput the name of a object file in the "resources/objects" directory ("cube.obj", "cyborg.obj", or "rock.obj") and press the "reload model" button.
